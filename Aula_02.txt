---- comandos ---- 

cd.. >  voltar pasta 

cd (nome da pasta) > acessar pasta 

cls - Crtl + L > limpar terminal

git init > iniciar repositorio git

git add (comando para adicionar os arquivos)

git status > ver o status do repositorio

git commit -m "Mensagem que deseja colocar" > commitar 

git log > registro dos logs 

git push > "empurrar" para o github as alteraçoes 

git pull > comando para atualizar repositorio local

git fetch > leva as informaçeos do antigo repositorio para conseguir fazer uma comparacao por ex.

git branch (nome da branch) > criar uma branch (como se fosse a linha do tempo )

git checkout > trocar o nome da branch 

git checkout master > git merge (nome da branch q foram feitas as alterações no codigo)

git branch > lista as branches
